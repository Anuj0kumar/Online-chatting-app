## basic chatting web app <br>
* you can chat we people who are on same network as you , going to implement end to end encription over time 

* please install required modules for working of the app

**Requirments**

```
 pip install flask flask-socket eventlet 
```

* if you are running it on your "localhost"  Other user with `your-localhost-ip:5000` link can join the chat (if they are on the same network as you )
